crearNodo(tagName, nodeText, nodeId, nodeClasses, nodeAttributes)

Crea un nodo en el DOM con la etiqueta HTML y los atributos que queramos asignarle.

Parameters:
Name Type Description
tagName String

Nombre de la etiqueta HTML que deseamos crear.
nodeText String

Atributo texto del elemento HTML.
nodeId String

Atributo id del elemento HTML.
nodeClasses Array

Nombre de las clases en un array. Ejemplo: ['parchita', 'velo'] Resultado: class="parchita velo".
nodeAttributes Array

Array de clave-valor de los atributos adicionales a añadir. Ejemplo: [{name: 'parchita', key: 'velo'}, {name:'crazy', key:'wombat'}] Resultado: parchita="velo" crazy="wombat".
Returns:
  • Nodo con la etiqueta y los atributos asignados.
Type
*

generarTemplate(objeto, containerID, templateTag)

Crea y añade nodos al documento HTML en base a los datos del objeto recibido, ya sea utilizando HTML templates o la función crearNodo(), cuando el navegador no sea compatible con las templates.

Parameters:
Name Type Description
objeto Object

Objeto con los datos que queremos renderizar en el DOM.
containerID *

ID del contenedor donde ubicaremos los nodos generados con templates.
templateTag *

Etiqueta HTML de la template que queremos utilizar.
